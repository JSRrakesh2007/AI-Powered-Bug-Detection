## Project Scope

This project automatically analyzes Python source code and detects syntax-related bugs. It uses Python's Abstract Syntax Tree (AST) module to identify coding errors and generate a bug report.

## Technologies Used

- Python
- AST Module

## Features

- Detects syntax errors
- Generates bug reports
- Saves results to a text file
- Easy to use

## How to Run

```bash
python bug_detector.py
```

## Output

The program displays detected bugs and saves them in bug_report.txt.
